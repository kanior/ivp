# IVP 영화 예매 사이트 이용 방법

- 접속 URL
> ec2-52-79-129-143.ap-northeast-2.compute.amazonaws.com:8080

- 회원가입 및 로그인 기능 이용 가능

- 영화 페이지, 극장 페이지 이용 가능

- 로그인시 예매 페이지 이용 가능 (관리자 계정으로 미리 영화 상영 스케쥴 등록해야 함)

- 관리자 계정
> ID : admin

> PW : admin1234

- 관리자 계정으로 로그인시 영화 상영 스케쥴 등록 페이지 이용 가능

- 로그인시 내정보 페이지 이용 가능

- 로그인 후 영화 예매시 내정보 페이지에서 예매내역 페이지 이용 가능
